Una de las cosas mas incómodas para mí con Godot ha sido iniciar un proyecto en github, ya que para iniciar un proyecto con Godot debo crear el proyecto en una carpeta en vacía, aunque Godot se integra bien con Git y Github al inicio mas que algo difícil puede llegar a ser un poco tedioso nada mas.

Por eso he aprovechado los templates de Github para poder iniciar proyectos mucho mas fácil y de una manera muy cómoda.

Creando le repositorio

Lo que debemos hacer es ir a este enlace y luego dar clic en Use this template luego nos enviará al formulario para crear nuestro repositorio en github.

 
 Luego de dar clic en Use this template te enviará al formulario:

El template actualmente funciona con Godot 3.1.2, aunque preveo actualizarlo para las futuras versiones.

Este template contiene el proyecto inicial de godot con template como nombre del proyecto, un .gitignore por defecto para Godot y un README.md.

Usar el repositorio

Para las mayoría de las personas familiarizadas con Git y Github los pasos siguientes son innecesarios pero si eres principiante en el tema te muestro cómo trabajar con este repositorio a continuación.

Clonar el repositorio

Después de crear el repositorio Github nos enviará al enlace de éste, ahí daremos clic en el botón de clone or download y copiaremos el enlace que aparece, importante: no vamos a descargar nada ahí, no tienes que dar clic en Download ZIP.

Si no tienes instalado Git no podrás seguir los pasos, puedes descargar Git con este enlace.

Abrimos la terminal o CMD en el caso de windows, y nos ubicamos en el directorio donde queramos colocar el proyecto, yo te recomiendo crear una carpeta Repositorios en Documentos, ante todo el orden. Luego con el comando cd nos podemos ubicar ahí, por ejemplo, escribiendo "cd Documentos/Repositorios" sin comillas.

Luego vamos a escribir: git clone enlace-que-copiamos-en-github.git y damos Enter.


Si el repositorio es público lo descargará a continuación, si es privado te pedirá poner el usuario y contraseña de tu cuenta de github.

Con esto ya tenemos el proyecto inicial de Godot listo para trabajar con Git y Github.

Importar el proyecto


Abrimos Godot y damos clic en Importar, luego en Examinar y buscamos el proyecto clonado, específicamente el archivo project.godot y seleccionamos Abrir.

 Actualizar el proyecto

Ya tenemos todo listo, podemos comenzar a trabajar en el juego y para finalizar el post, vamos a explicar un poco cómo actualizar el proyecto en github.

Cuando ya tengas tu primer avance y quieras subir tu avance a github, debes hacer lo siguiente:

Abrir la terminal o CMD y ubicarte en la carpeta de tu repositorio (Documentos/Repositorios/tu-juego).

Escribe el comando: "git add ." Todo sin comillas.

Escribe el comando: (git commit -m "descripcion de tu actualización") Todo sin paréntesis, lo que está entre comillas sería un comentario sobre los cambios en tu proyecto.

Escribe el comando: "git push", sin comillas y luego te pedirá tu usuario y contraseña de tu cuenta de github, debes tener internet para realizar este paso obviamente.
