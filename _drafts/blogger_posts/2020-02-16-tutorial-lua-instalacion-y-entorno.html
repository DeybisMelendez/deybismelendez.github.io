Instalación en Windows

Para instalar Lua en Windows solo ve al enlace http://luabinaries.sourceforge.net/download.html y elige win32 o win64 según la arquitectura de tu pc, los archivos que descargues debes colocarlos en la raíz del disco local C.

Instalación en Linux

Para instalar en linux abre la terminar y ejecuta el comando:
sudo apt install lua5.3

Una vez instalado podemos ejecutar el CMD o Terminal según el caso: 
lua -v

Nos debería imprimir la versión de Lua que tenemos instalado.

Instalar un IDE o Editor de Código

Los editores de códigos que puedes usar son Atom, VS Code, Zerobrane Studio, incluso Notepad++, en el caso de Atom y VS Code hay que descargar plugins de Lua para tener una máxima experiencia programando.

Personalmente recomiendo Atom o VS Code.
