Nano es un ligero editor de texto que se usa desde una terminal, es muy útil para editar archivos sin necesidad de abrir un editor gráfico. Viene preinstalado en la mayoría de distribuciones linux, a diferencia de Vim, Nano es mas simple y su curva de aprendizaje es mas rápido, en este artículo te mostraré como configurar Nano para hacerlo mas amigable al editar.

Primero vamos a crear el archivo de configuración. Este es un archivo llamado ".nanorc" que debe ubicarse en la carpeta de usuario, /Home/Usuario/, para eso podemos ejecutar en terminal lo siguiente:
touch ~/.nanorc
Ahora podemos editarlo con el siguiente comando:
nano ~/.nanorc
Ahora nos va a aparecer el editor de texto nano.

Puedes presionar Ctrl+x para cerrar Nano, luego presionas Y para guardar, N para no guardar, y Ctrl+C para cancelar.

Recuerda que siempre que veas ^ significa la tecla Ctrl.

Ahora toda la configuración la podemos colocar ahí.

Ahora podemos configurar el archivo .nanorc, solo lo abres, y escribes:

Para añadir número de lineas:
set linenumbers
Para poder usar el mouse para moverte en el texto:
set mouse
Para autoindentar el texto:
set autoindent
Para configurar el tamaño de la tabulación, (4) es el número de espacios:
set tabsize 4
Para convertir los tabs a espacios:
set tabstospaces
De manera que en total, el documento sería:
set linenumbers
set mouse
set autoindent
set tabsize 4
set tabstospaces
Al final tendremos a Nano mas amigable para utilizar.
Adicionales

Puedes configurar los colores de la interfaz:

El título:
set titlecolor foregroundColorCode,backgroundColorCode
foregroundColorCode es el color de frente, el del texto y backgroundColorCOde el color de fondo.

Los colores soportados son white, black, blue, green, red, cyan, yellow, magenta.

Es decir, podemos escribir:
set titlecolor black,green
Tendremos algo así:
Tambien tenemos los siguientes:

statuscolor: cambia el color del texto que indica el estado del documento, si le cambiamos al igual que el título quedaría así:
numbercolor cambia el color del número de línea y keycolor cambia el color del comando.

Puedes ver mas comandos de configuración aquí: https://www.nano-editor.org/dist/v2.1/nanorc.5.html
