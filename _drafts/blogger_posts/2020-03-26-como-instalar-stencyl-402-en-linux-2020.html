Stencyl fue el primer engine que usé para crear videojuegos, es un engine ideal para los que quieran iniciar sin tener conocimientos en programación y pronto iniciará el Stencyl Jam #19, por lo que he decidido participar, sin embargo yo actualmente uso Linux y el tutorial que se encuentra en la web no está actualizado, así que rebuscando por internet encontré la forma de instalarlo.

Primero ejecutamos lo siguiente:

sudo apt-get install g++ libgc-dev libxext-dev
sudo dpkg --add-architecture i386
sudo apt-get update
sudo apt-get install libxtst6:i386 libxi6:i386 libncurses5:i386 libxt6:i386 libxpm4:i386 libxmu6:i386

Luego descargamos Stencyl desde la web: http://www.stencyl.com/download/

Descomprimimos y colocamos la carpeta en un sitio donde queramos tener el engine, y luego abrimos la terminal en esa carpeta y ejecutamos:

sudo ./Stencyl

Listo, desde este momento podrás crear y probar tus juegos en Stencyl, para probar y exportar te recomiendo que uses HTML ya que es el mas rápido y en formato Flash no hay compatibilidad, ejecutar con sudo es importante para poder exportar, de lo contrario lo tendrá desactivado.
